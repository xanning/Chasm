# Chasm
Enhance your vision with this X-Ray mod for Minecraft 1.8.9 Forge (bypasses hypixel!). Ported from Myau.

## Usage

### Commands

All commands also support tab completion.

| Command | Description |
|---------|-------------|
| `/xray` | View current settings. |
| `/xray on` | Enable Xray |
| `/xray off` | Disable Xray |
| `/xray bind <key>` | Bind to Toggle Xray |
| `/xray mode <soft\|full>` | Set rendering mode (SOFT = translucent, FULL = full x-ray) |
| `/xray opacity <0-100>` | Set block opacity percentage |
| `/xray range <16-512>` | Set ore detection range |
| `/xray caves-only <true\|false>` | Only show ores exposed to caves (uhc bypass) |
| `/xray caves-radius <1\|2>` | Cave detection radius |
| `/xray diamonds <true\|false>` | Toggle diamond ore tracking |
| `/xray diamonds-tracers <true\|false>` | Toggle diamond tracers |
| `/xray gold <true\|false>` | Toggle gold ore tracking |
| `/xray gold-tracers <true\|false>` | Toggle gold tracers |
| `/xray iron <true\|false>` | Toggle iron ore tracking |
| `/xray iron-tracers <true\|false>` | Toggle iron tracers |
| `/xray coal <true\|false>` | Toggle coal ore tracking |
| `/xray coal-tracers <true\|false>` | Toggle coal tracers |
| `/xray redstone <true\|false>` | Toggle redstone ore tracking |
| `/xray redstone-tracers <true\|false>` | Toggle redstone tracers |
| `/xray lapis <true\|false>` | Toggle lapis ore tracking |
| `/xray lapis-tracers <true\|false>` | Toggle lapis tracers |
| `/xray emeralds <true\|false>` | Toggle emerald ore tracking |
| `/xray emeralds-tracers <true\|false>` | Toggle emerald tracers |
| `/xray spawners <true\|false>` | Toggle spawner tracking |
| `/xray spawners-tracers <true\|false>` | Toggle spawner tracers |
| `/xray canes <true\|false>` | Toggle sugar cane tracking |
| `/xray canes-tracers <true\|false>` | Toggle sugar cane tracers |
| `/xray warts <true\|false>` | Toggle nether wart tracking |
| `/xray warts-tracers <true\|false>` | Toggle nether wart tracers |
